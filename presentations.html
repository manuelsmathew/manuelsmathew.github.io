Enhanced Diagnostics for Wind Turbine Pitch System

PHME 2024 - Prague, Czech Republic

Introduced novel algorithm to detect and localize faults in induction motors in a wind turbine pitch system using SCADA data. Our deep learning approach significantly improves fault detection accuracy and enables proactive maintenance strategies.

Wind Farm Performance Estimation using Deep Learning

PHME 2024 - Prague, Czech Republic

Presented a comprehensive deep learning architecture to estimate the wind farm level performance using SCADA data. This research enables more accurate performance monitoring and optimization strategies for large-scale wind energy installations.
